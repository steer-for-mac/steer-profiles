# Steer Profiles

Community-curated Steer profiles for popular Mac apps.

## Import a Profile

1. Open the profile JSON in `profiles/`.
2. Copy the raw GitHub URL.
3. In Steer, open Settings > Profiles and choose Import from URL.

## Contribute

Add one `.json` file per app bundle ID under `profiles/`, then open a pull request. The validation workflow checks every profile document before review.
